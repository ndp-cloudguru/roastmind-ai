# RoastMind AI ☕

AI Coffee Concierge & Automation Assistant

---

# Overview

RoastMind AI is an AI-powered coffee recommendation and automation platform designed for:

* Specialty coffee shops
* Coffee roasteries
* Coffee lovers

The system provides:

* AI coffee recommendation
* Customer profile memory
* Marketing content generation
* Workflow automation
* Customer follow-up automation

---

# Tech Stack

## Frontend

* Nuxt.js
* TailwindCSS

## Backend

* Golang Fiber

## AI & Automation

* OpenAI API
* n8n Workflow

## Database

* Supabase / PostgreSQL

## Hosting

* Vercel
* Railway

* AI Coffee Chatbot
