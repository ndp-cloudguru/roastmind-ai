# Công việc đã thực hiện

- Tìm hiểu Specialty Coffee Market
- Phân tích hành vi khách hàng
- Nghiên cứu mô hình vận hành quán cà phê

## 2. Phân tích bài toán

Các vấn đề phổ biến:

- Khách hàng khó chọn cà phê phù hợp
- Quán thiếu trải nghiệm cá nhân hóa
- Marketing tốn thời gian
- Thiếu chatbot hỗ trợ khách hàng

## 3. Nghiên cứu AI Solutions

Đã nghiên cứu:

- ChatGPT API
- Claude API
- AI chatbot
- Automation workflow
- Recommendation system

## 4. Lựa chọn Tech Stack

Nhóm quyết định sử dụng stack đơn giản, phù hợp MVP:

| Thành phần | Công nghệ |

| Frontend | HTML/CSS hoặc Google Sites |
| AI Engine | ChatGPT API |
| Automation | n8n |
| Database | Google Sheets / Supabase |
